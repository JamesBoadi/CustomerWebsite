2. Advanced javascript

Objects

Objects in this case are used for every piece of game that has to provide information or interact with other elements. The reason they are used is to define how certain objects behave in a simple and easily understandable way, for example projectile or tank objects within the game contain all of their behaviour in one prototype function (projectile has gravity and directional movement properties, tank object has specific gun movement, etc.) which makes everything easier to understand, update and maintain. Code location: game.js file, lines: 138-300.

Canvas

Canvas element and manipulation of it in javascript is the base of this game. Animation, physics and graphics were visualised and implemented using drawing functionality canvas element provides. For example object movement is created by drawing and redrawing each frame differently according to how it interacts with the suroundings.
