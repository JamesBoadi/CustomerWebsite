HTML Structure elements

1. Nav

Nav element is mainly used for holding navigation bar within every page of this website. The code can be found in function.php, lines: 32-48.

2. Article

Article element is used within Home page to contain Home page content. The code can be found in index.php, lines: 13-28.

3. Header

Header element is used in Home page to function as a "Welcome" and "How to play" text conatiner. The code can be found in index.php, lines: 15-18.

4. Section

Section element conatins the list of commands that will be used to interact with the game. The code can be found in play.php, lines: 17-25.

5. Footer

Footer element holds the bottom of the page content - Middlesex logo, and is used in every page. The code can be found in functions.php, lines: 54-59.

HTML Content elements

1. h1

H1 element is used in Home and Highscores pages for "Welcome" text and "Leaderboard" title.

2. h2

H2 element is used in Home page after "Welcome" text to contain the "How to play" text.

3. h4

H4 element is used in Highscores page to hold "View full leaderboard" link.

4. a

A element can be found within navigation bar, as well as on all the logos and "View full leaderboard" link.

5. div

Div element is within every page as a conatiner for grid layout and as a container for Login/Signup area and Leaderboard.

6. ul

Ul element is used to display "How to play" content.

7. ol

Ol element is used to display Leaderboard in numerical order inside Highscores page.

8. li

Li element is used to conatin list items within every Ol, Ul and Menu element in the website.

9. menu

Menu element conatins links in navigation bar.

10. img

Img content element is used to display Logo of the website as well as Middlesex logo in the footer.

11. input

Input element can be found in Login/Signup page within login and signup areas.

12. button

Button element is used in login and signup areas.

CSS Page design

Text

Text formating is applied with the use of font: ('Courier New', monospace), and text spacing which is set to 1 px. Font and text spacing can be found in styles.css file within styling block of body element, style.css Lines: 10-11.

Colour

Colour formating is applied on navigation bar and button elements. When hovering the background and text colour changes. Colour formatting is also applied with .active class elements in navigation bar. Navigation bar and content of each page has opacity property set to them in order to make them transparent.

Positioning

Position formatting is mostly applied with margins, grid layout and flex properties. Navigation bar is made using inline-flex and justify-content: space-between properties to align and separate logo and links. Width property and Margins are used on every page in order to center elements. Every page has a seperate div element with grid layout, to dedicate a place for specific part of the page content styles.css, lines: 72-79, 109-116, 218-226.

Shape

Shape formating is applied with the use of rounded corners. Content in every page has border-radius property applied, that includes Canvas element in Play.php page, Leaderboard in highscore.php page and content inside Home page.

CSS Classes and IDs

Class

Classes are used for navigation bar as .navBar in styles.css lines: 23-67, because in every page navigation bar has to appear almost the same. Class .active is used to highlight elements in navigation bar which are currently visible to the user, styles.css lines: 58-61. Another use of class is with login and signup forms. Both areas are styled the same therefore class .inputfield is used on both, styles.css lines: 180-214.

ID

ID is used with all page containers of the website, because each page requires slighlty different grid layout, styles.css lines: 72-79, 109-116, 218-226, 275-282. ID is as well used for the websites logo, because it needs individual styling and is only used once. IDs are assigned to each input field inside login and signup areas, they need to have IDs in order to access their values later on in the second part of the project, loginregister.php lines: 24, 26, 33, 35, 37.

Grid layout

Every page container has an individual ID assigned in order to allow specific grid layout to be applied. Display: grid; property is used, and the place of each part of the page content is sorted by grid-template-areas property, styles.css lines: 72-79, 109-116, 218-226, 275-282.

PHP

PHP functions

Function navBar($pageName) generates a navigation bar, assigns ID with grid layout to a specified page, and sets class to .active to the page that is currently displayed. It is called at the top of every page after the function outputHeader($title). Function can be found in the functions.php file within the lines: 20-49.

Reusable PHP functions

Function outputHeader($title) uses echo command to output , opening html tag as well as the whole head including the title of the page and opening body tag. It is called at the top of every page. Function can be found in functions.php file, lines: 5-16.

Function footer() uses echo command to output the footer with Middlesex logo and closing body and html tags. It is called at the bottom of every page. Function can be found in functions.php file, lines: 53-63.
